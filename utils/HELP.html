---BOT CONTROLS

Bot Mode: Select if you're fishing, mining, or just battling pokemon by walking.

New Moves: When a pokemon is learning a new move which slot should it go to, if any. If you have available empty slots those will get filled up first.

Evolve: Select yes if you would like to accept pokemon evolutions.

Avoid Elite: Select yes if you would like to run away from elite pokemon.

Heal %: Select HP threshold when a potion should be used on your pokemon. When a pokemon is used in a catch rule that pokemon will always be healed to 95% at least as a preacution.

Speed: How fast you would like the bot to run. The default is 3, higher you go the more likely you are to get banned.

Catch List: Pokemon you would like to catch.

Avoid List: Pokemon you would liek to run away from. Good to use for EV training. Make sure you don't have the same pokemon in catch list and avoid list if you want this to work.

---HOW TO FISH

  • You must be loged in and facing water.
  • You must have a rod in your inventory.
  • Change Bot Mode to "Fish" from the drop-down.
  • Configure catch/avoid list and any other settings as you'd like.
  • Press start.

---HOW TO MINE

  • You must be logged in and standing next to one or more rocks.
  • You must have a pickaxe in your inventory.
  • Change the Bot Mode to "Mine" from the drop-down.
  • Configure catch/avoid list and any other settings as you'd like.
  • Press start.

---HOW TO BATTLE

  • You must be logged in and in an area with Pokemon. Grass, water, cave, etc.
  • Under Bot Control click the Select Mode button.
  • Select an area on the map where you would like to hunt for pokemon. You must select at least 4 tiles but more is recommended. Select tiles around and including the one you're standing on, otherwise your character will not move.
  • Change the Bot Mode to "Battle" from the drop-down.
  • Configure catch/avoid list and any other settings as you'd like.
  • Press start.
